## 第一部分：你在做什么？

### 核心问题
你是一个中国卖家，在亚马逊/eBay/独立站上卖东西给德国人。但你遇到了三个大问题：

1. **红人太多，真假难辨** - 想找德国KOL做推广，但不知道谁是真红人，谁是刷粉机器

## 第二部分：德国市场为什么特殊？

### 德国消费者的三大特点

#### 1. 极其讨厌虚假宣传 (Authentizität)
- **现象**：德国人对"刷粉红人"零容忍
- **后果**：如果你找的红人被发现是刷粉，品牌信誉瞬间崩塌
- **你的工具**：红人评估器会检测互动率、粉丝质量，过滤掉僵尸粉账号

**例子**：
```
❌ 坏红人：10万粉，但每条帖子只有200赞（0.2%互动率）
✅ 好红人：5000粉，但每条帖子有500赞（10%互动率）
```

#### 2. 极其讨厌"一次性消费品" (Langlebigkeit)
- **现象**：德国人买东西不看便宜，看能用多久
- **后果**：你的产品描述必须强调"耐用性"，不能只说"便宜"
- **你的工具**：关键词匹配会自动识别红人是否关注"质量"而非"价格"

**例子**：
```
❌ 错误方向：强调"只要99欧！"
✅ 正确方向：强调"用了5年还没坏，德国制造工艺"
```

#### 3. 极其讨厌"不透明" (Transparenz)
- **现象**：德国人要求知道产品从哪来、怎么做的、材料是什么
- **后果**：你的邮件必须清楚说明公司信息、产品来源
- **你的工具**：Privacy_Check会自动检查你的邮件是否包含法律要求的公司信息

**例子**：
```
❌ 违法邮件：只有品牌名，没有公司地址、电话、税号
✅ 合法邮件：包含公司名、地址、电话、税号、法定代表人
```

## 第三部分：你的三个工具怎么用？

### 工具1：红人评估器 (Influencer Evaluator)

**问题**：我想找德国时尚博主推广我的包包，但不知道谁是真红人

**解决方案**：
```python
# 输入：红人的Instagram数据
profile = {
    "followers": 50000,
    "avg_likes": 2500,
    "avg_comments": 150,
    "bio": "Nachhaltige Mode aus Berlin 🌿",  # 可持续时尚
    "recent_posts": ["我的新包包", "环保材料", "德国制造"]
}

# 输出：评分 + 建议
评分: 94/100 (S级)
建议: 强烈推荐！这个红人关注可持续性，粉丝互动健康
```

**三维度评分**（权重）：
- **真实性 45%** ⭐⭐⭐⭐⭐ 最重要！
  - 检查：互动率是否正常？粉丝是否真实？
  - 红旗：互动率<1%、粉丝突然增长、评论都是机器人
  
- **相关度 35%** ⭐⭐⭐⭐
  - 检查：红人的内容是否与你的产品匹配？
  - 例子：你卖环保包包，红人经常发"可持续时尚"内容 = 高相关度
  
- **活跃度 20%** ⭐⭐⭐
  - 检查：红人多久发一次帖子？
  - 注意：德国顶级博主发帖不频繁，但每篇都是深度内容

**问题**：我想给这个红人发邮件，但不知道怎么写才不违反德国法律

# 输入：红人信息 + 产品信息
context = {

合规检查: ✅ 包含退订选项 ✅ 包含数据保护声明 ✅ 包含公司信息

**两种模式**：

1. **严谨商务模式 (Formal)**
   - 用于：第一次联系、大品牌、B2B
   - 语气：正式、专业、尊重
   - 例子：
   ```
   Sehr geehrte Frau Schmidt,
   ich bin auf Ihr Profil aufmerksam geworden und war beeindruckt 
   von Ihrer Arbeit im Bereich nachhaltige Mode.
   ```

2. **社交媒体亲和模式 (Friendly)**
   - 用于：TikTok、年轻博主、第二次跟进
   - 语气：轻松、亲切、有emoji
   - 例子：
   ```
   Liebe Anna,
   ich liebe, wie du fashion Themen rüberbringst – echt authentisch! 🙌
   ```

**法律合规自动检查**：
- ✅ 退订选项 (UWG §7) - 德国反垃圾邮件法
- ✅ 数据保护声明 (GDPR Art.13) - 欧盟隐私法
- ✅ Impressum (TMG §5) - 德国公司信息披露法

### 工具3：差评风险检测 + 道歉信生成

**问题**：我收到一条1星差评说"要告律师"，怎么办？

**解决方案**：
```python
# 输入：差评内容
review = "Ich werde meinen Anwalt einschalten! Das ist Betrug!"

# 输出：风险等级 + 自动生成的道歉信
风险等级: 🔴 CRITICAL (紧急)
原因: 检测到法律威胁关键词 "Anwalt" (律师)
建议: 立即通知法务团队，24小时内电话联系客户

自动生成的道歉信:
主题: Dringende Angelegenheit - Bestellung ORD-12345
正文: [自动生成的德语道歉信]
补偿建议: 全额退款 + 20%优惠券
```

**四类风险等级**：

1. **🔴 CRITICAL (紧急)** - 立即处理
   - 关键词：Anwalt (律师), Verbraucherzentrale (消费者保护中心), Betrug (欺诈)
   - 后果：可能被起诉、被投诉到政府部门
   - 行动：24小时内电话联系客户，准备法务审核

2. **🟠 HIGH (高风险)** - 24小时内处理
   - 关键词：Rückerstattung (退款), defekt (坏了), kaputt (破了)
   - 后果：可能被要求退款、退货
   - 行动：准备退款/换货流程

3. **🟡 MEDIUM (中等)** - 3天内处理
   - 关键词：不满意、质量一般、价格贵
   - 后果：影响店铺评分
   - 行动：发送友好的回复，提供小补偿

4. **🟢 LOW (低风险)** - 正常回复
   - 内容：正常的建议、中立评价
   - 后果：无
   - 行动：感谢反馈，正常回复

## 第四部分：德国法律为什么这么严格？

### 三大法律红线

#### 1. TMG §5 (Telemediengesetz) - 公司信息披露
**要求**：任何商业邮件必须包含：
- ✅ 公司名称
- ✅ 公司地址
- ✅ 电话号码
- ✅ 邮箱地址
- ✅ 增值税号 (USt-IdNr)
- ✅ 商业登记号 (Handelsregister)
- ✅ 法定代表人姓名

**后果**：违反 = 罚款 €5,000-€50,000

**你的工具**：Privacy_Check 会自动检查这些

#### 2. UWG §7 (Gesetz gegen den unlauteren Wettbewerb) - 反垃圾邮件
**要求**：
- ✅ 必须包含"退订"选项
- ✅ 不能隐藏退订按钮
- ✅ 退订必须有效

**后果**：违反 = 罚款 €5,000-€300,000

**你的工具**：开发信生成器自动添加退订选项

#### 3. GDPR (General Data Protection Regulation) - 隐私保护
**要求**：
- ✅ 必须说明如何使用客户数据
- ✅ 必须说明数据不会被转卖
- ✅ 客户有权要求删除数据

**后果**：违反 = 罚款最高 €20,000,000 或营收的4%

**你的工具**：开发信生成器自动添加数据保护声明

## 第五部分：关键词为什么重要？

### 德国消费者的价值观关键词

#### 优先级1：可持续性 (Nachhaltigkeit) ⭐⭐⭐⭐⭐
**为什么**：德国人极其关心环保
**关键词**：
- nachhaltig (可持续的)
- umweltfreundlich (环保的)
- öko (生态的)
- recycling (回收)
- plastikfrei (无塑料)
- fair trade (公平贸易)

**例子**：
```
❌ "我们的包包很便宜"
✅ "我们的包包用100%回收皮革制造，完全可持续"
```

#### 优先级2：可靠性/质量 (Zuverlässigkeit/Qualität) ⭐⭐⭐⭐
**为什么**：德国人买东西看质量，不看便宜
**关键词**：
- qualität (质量)
- zuverlässig (可靠的)
- hochwertig (高档的)
- langlebig (耐用的)
- robust (坚固的)
- made in germany (德国制造)

**例子**：
```
❌ "只要99欧！"
✅ "德国制造，用了5年还没坏，质量保证"
```

#### 优先级3：透明度 (Transparenz) ⭐⭐⭐
**为什么**：德国人要知道产品从哪来、怎么做的
**关键词**：
- transparent (透明的)
- herkunft (来源)
- material (材料)
- handwerk (手工艺)
- prozess (过程)

**例子**：
```
❌ "我们的产品很好"
✅ "我们的产品在意大利手工制造，使用西班牙皮革，
   所有材料来源都可追溯"
```

#### 优先级4：性价比 (Preis-Leistung) ⭐⭐
**为什么**：德国人不是不在乎价格，而是要"值"
**关键词**：
- preis-leistung (性价比)
- wert (价值)
- investition (投资)
- langfristig (长期的)

**例子**：
```
❌ "便宜！"
✅ "虽然价格较高，但能用10年，平均每年只要10欧"
```
